# 🧮 Calculator - Keyboard Support

A simple and responsive calculator built using **HTML, CSS, and JavaScript** with full keyboard support.

---

## 🚀 Features
- Basic operations (+, −, ×, ÷)
- Keyboard input support
- Clear (C) and Backspace (⌫)
- Responsive UI using CSS Grid

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 🎯 How to Run
1. Download ZIP from GitHub
2. Extract the folder
3. Open `index.html` in any browser

---

## ⌨️ Keyboard Controls
- Numbers → 0–9
- Operators → +, -, *, /
- Enter → Calculate result
- Backspace → Delete last value
- Escape → Clear display
